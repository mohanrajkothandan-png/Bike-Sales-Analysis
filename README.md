# 🚴 Bike Sales Analysis Dashboard

## 📖 Project Overview

This project is an end-to-end sales analysis built in **Microsoft
Excel** to explore customer purchasing behavior and business performance
in the bike market. Using Pivot Tables, Pivot Charts, and an interactive
dashboard, the project transforms raw data into meaningful insights that
support data-driven decision making.

------------------------------------------------------------------------

## 🎯 Objectives

-   Analyze bike purchase patterns across different customer segments.
-   Understand how age, income, occupation, and other demographics
    influence sales.
-   Build an interactive dashboard for quick business reporting.
-   Demonstrate practical Excel skills for data analytics portfolios.

------------------------------------------------------------------------

## 🛠️ Tools & Features

-   **Microsoft Excel**
-   **Pivot Tables**
-   **Pivot Charts**
-   **Slicers**
-   **Interactive Dashboard**
-   **Data Cleaning & Preparation**
-   **Conditional Formatting**

------------------------------------------------------------------------

## 📊 Dashboard Highlights

The dashboard enables users to: - View overall bike purchase trends. -
Compare purchases by age group and gender. - Analyze customer income
distribution. - Explore demographic patterns interactively using
filters. - Identify business insights through visual summaries.
